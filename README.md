# scrum-practice-bootcamp
Practicing GitHub environment setup using the Projects feature and simulating an agile/Scrum workflow."

### Process Documentation
For this simulation, we selected Google Colab as our primary analysis tool due to its cloud-based environment and seamless integration with GitHub. The connection was established directly through Colab’s native interface, allowing for efficient version control. One of the main challenges encountered during the setup was mapping the technical workflow in the notebook to the specific task cards within GitHub Projects. Overcoming this required a clear definition of how each code module contributed to the deliverables defined in the Product Backlog.
